This is our Khidmat Project for Al-Khidmat Aghosh Homes. The project consists of a working prototype of the Aghosh Home Orphanage Management System, including all major modules such as: 
o Orphan profiles management 
o Staff/volunteer management 
o Donation tracking system 
o Events & announcements 
o Attendance & daily activity records 
o Secure login & role-based access 

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
